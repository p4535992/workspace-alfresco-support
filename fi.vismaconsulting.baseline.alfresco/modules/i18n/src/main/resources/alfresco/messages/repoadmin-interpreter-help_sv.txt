##
##  Meta kommandon
##

ok> help

    Listar denna hjälp

ok> r

    Repetera senaste kommando.
    
##
##  Quit / Exit
##

    Om du kör en webbläsare, stäng detta fönster eller tab (och logga ut från Alfresco Explorer eller stäng webbläsare).
    
    Om du kör kommand-radsverktyg, skriv 'quit' eller 'exit'.

##
##  Generalla Repo Admin Kommandon
##

ok> show file <fileClassPath>

    Visar innehåller för filen i sökvägen <fileClassPath>.

    <fileClassPath> Sökvägen till en fil

    Till exempel show file alfresco/extension/xxxModel.xml
    Till exempel show file alfresco/extension/yyy-messages.properties
    
ok> show file-list <filesClassPath>

    Visar en lista på filer i sökvägen <filesClassPath> med en första matchning för varje filnamn.

    <filesClassPath> sökväg till en lista av filer. Wildcard * är tillåtet. Till exempel. för att se
    en lista på resource bundles som laddas så använd: /path1/path2/bundlename*.properties

    Till exempel show file-list alfresco/extension/*
    Till exempel show file-list alfresco/extension/*Model.xml
    Till exempel show file-list alfresco/extension/zzz-messages*.properties
        
##
##  Model Admin Kommandon
##

ok> show models

    Visa modeller - som lagras i repository data dictionary.

ok> deploy model <fileClassPath>

    Ladda upp modell till repository och ladda i runtime data dictionary. Det här kommer också
    repository modellen som aktiv. 
    
    Om en modell redan är publicerad (deployed) kommer den att uppdateras och återpublicers (re-deployed).

    Till exempel deploy model alfresco/extension/exampleModel.xml

ok> undeploy model <modelFileName>

    Permanent radera en modell från repository (all versioner) ladda ur från runtime data dictionary.

    Till exempel undeploy model exampleModel.xml

ok> activate model <modelFileName>

    Sätt repository modellen till aktiv och ladda in till runtime data dictionary.
    
    Till exempel activate model exampleModel.xml
    
ok> deactivate model <modelFileName>

    Sätt repository modellen till inaktiv och ladda ur från runtime data dictionary. 
    
    Till exempel deactivate model exampleModel.xml
    
##
##  Message Admin Kommandon
##

ok> show messages

    Visa meddelande resource bundles - som lagras i repository data dictionary.

ok> deploy messages <resourceBundleClassPath>

    Ladda upp meddelande resource bundle till repository och runtime message service.

    Till exempel deploy messages alfresco/extension/lifecycle-messages
    
ok> undeploy messages <resourceBundleBaseName>

    Ta bort meddelande resource bundle från repository och runtime message service.
    
    Till exempel undeploy messages lifecycle-messages
    
ok> reload messages <resourceBundleBaseName>

    Läs om meddelande resource bundle från repository och runtime message service.
    
    Till exempel reload messages lifecycle-messages
